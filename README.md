# Location de voiture – Laravel 🚗

Projet Laravel simple de gestion de location de voitures.

## Fonctionnalités
- Ajouter une location
- Afficher la liste des locations
- Supprimer une location
- Calcul automatique du nombre de jours et du total

## Technologies
- Laravel
- PHP
- MySQL
- HTML / CSS
